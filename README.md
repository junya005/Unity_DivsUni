# DivsUni

「DivsUni」は、迫りくる弾を避けながらスコアを稼ぐ、カジュアルな避けゲーム（サバイバルアクション）です。
専門学校の1年目に、個人制作（開発人数：1人）として開発した作品となります。

## 🎮 ゲーム概要

様々な種類の弾（通常弾、直進弾など）を避けながら、同時に「酸素（O2）」を管理して生き残るゲームです。
単なる避けゲーに留まらず、O2ゲージの残量というリソース管理の要素を組み合わせることで、プレイヤーに状況判断を要求するゲームデザインとなっています。

## ✨ 主な機能と特徴

- **O2（酸素）管理システム**
  - 時間経過などで減少するO2ゲージを管理しながら立ち回る必要があります。
  - 関連スクリプト: `O2GageController`, `O2Generator`, `o2Controller`
- **多様な敵と弾の挙動**
  - 複数のパターンの弾を実装し、避け方に変化を持たせています。
  - 関連スクリプト: `EnemyController`, `NomalBullet`, `StraightBullet`, `DeleteBullet`
- **スコア記録システム**
  - ゲームのスコアを保存・読み込みする機能を実装しています。
  - 関連スクリプト: `ScoreDataClass`, `ScoreSaveLoad`, `SC_ScoreDataBase`
- **ゲームサイクル管理とサウンド**
  - シーン遷移、ポーズ機能、オーディオ管理といった、ゲームの基本的な枠組みを構築しています。
  - 関連スクリプト: `GameManager`, `ResultManager`, `SceneLoader`, `GamePause`, `SoundManager`

## 🛠 開発環境・技術

- **ゲームエンジン**: Unity
- **プログラミング言語**: C#
- **開発体制**: 1人（個人制作）
- **制作時期**: 専門学校 1年目
